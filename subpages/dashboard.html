The Relaxy! Dashboard

dashboard.relaxy.xyz · building

A visual web control panel for Relaxy! Configure everything for your server - tickets, forums, heartboards, moderation, music. All without memorising a single command. It's live and evolving, so don't expect it to be perfect yet!

Take an early look

Heads up: this is a work in progress so things will change and break often. To resolve your issues message me @mattfor on discord or write to me on the matrix server

The idea

Relaxy! is all powerful. It shouldn't be hard to use, I want to make it easy and accessible.

The bot is "function over form"; deep, flexible, and admittedly a lot to learn. The dashboard is the solution, change everything you'd want to visually, not with strict commands

Everything, visually

Set up tickets, forums, heartboards, warnings and censoring with forms and toggles instead of long command chains.

Live server insight

See what Relaxy! is doing in your server at a glance; active systems, recent moderation, and health of every feature.

Safer changes

Previews and sensible defaults mean you can tweak configuration confidently, without fear of a typo breaking something.
