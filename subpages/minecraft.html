Why beta?

Old-school Minecraft, modern mod stack

Beta 1.7.3 is the version a lot of people remember most fondly. Beta Unleashed 2.0 keeps that feel and stacks tech, worldgen and quality-of-life mods on top.

Beta Unleashed 2.0

A curated Beta 1.7.3 modpack built on the Babric/StationAPI ecosystem. Everything is bundled, simply grab it, launch it, and you match the server.

Same roof as Relaxy!

The world runs on the very same Pi as the bot, the site, the CDN and the Matrix server; all self-hosted, all free to use.

Bring friends

It's a small community server for messing around in classic beta. Ask in the Discord if you'd like a hand getting connected.

Get started

Three steps to spawn in

  • Grab a launcher that supports old versions, like Prism Launcher.
  • Install Beta Unleashed 2.0 by following the modpack's README so your mods match the server.
  • Add a server pointing at minecraft.relaxy.xyz, hit join, and enjoy classic beta survival!
