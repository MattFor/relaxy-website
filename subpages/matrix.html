Why run my own?

Chat that actually belongs to you

Matrix is an open standard for real-time chat. Because I host the server myself, nothing about it answers to any company (or even me).

Yours by default

Every message lives on the server, not on someone's ad platform. No tracking, no data resale, no algorithm deciding what you see.

End-to-end encrypted

Private rooms are encrypted so that not even I can read them. Your keys stay on your devices.

Same roof as Relaxy!

One machine, one owner. The chat server sits right next to the bot, the site and the CDN - all self-hosted, all free.

Get started

Three steps to say hi

  • Open matrix.relaxy.xyz/login and register a new account.
  • Log in there in the browser, or point any Matrix client (like FluffyChat, Element) at the homeserver https://matrix.relaxy.xyz.
  • Find me, join a room in Relaxy!'s Fanclub and start chatting! Ask in the Discord support server if you'd like an invite.
